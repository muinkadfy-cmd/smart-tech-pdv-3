# ✅ CHECKLIST DE INSTALAÇÃO E ONBOARDING

**PDV Smart Tech - Guia para Implantação no Cliente**  
**Versão:** 1.0.0  
**Tempo estimado:** 1-2 horas

---

## 🎯 OBJETIVO

Este checklist garante que o sistema será instalado e configurado corretamente no cliente, evitando problemas futuros e garantindo satisfação.

---

## 📋 FASE 1: PRÉ-INSTALAÇÃO (Antes de ir ao cliente)

### ✅ Preparação

```
□ Confirmar agendamento com cliente (data, hora, local)
□ Confirmar requisitos técnicos:
  □ Computador/tablet disponível
  □ Sistema operacional compatível (Windows 7+, macOS, Linux)
  □ Navegador atualizado (Chrome, Edge, Firefox)
  □ Internet (se for usar sincronização na nuvem)
□ Levar material físico:
  □ Pen drive com backup de instalação
  □ Manual do usuário impresso
  □ Contrato/termo de uso
  □ Cartão de visita/contato de suporte
□ Preparar credenciais:
  □ URL do sistema
  □ Usuário admin padrão
  □ Senha temporária
```

---

## 📋 FASE 2: INSTALAÇÃO TÉCNICA (30-45 min)

### ✅ 1. Acesso ao Sistema

```
□ Abrir navegador (preferencialmente Chrome)
□ Acessar URL: ___________________________
□ Verificar que tela de login aparece corretamente
□ Fazer login com credenciais de admin
□ Confirmar acesso bem-sucedido
```

### ✅ 2. Configuração Inicial Obrigatória

```
□ Menu → Configurações → Empresa
  □ Nome da empresa
  □ Razão social (ou nome completo)
  □ CNPJ/CPF
  □ Telefone (aparece no recibo)
  □ Endereço completo
  □ Salvar

□ Menu → Configurações → Impressora
  □ Testar impressão
  □ Se térmica (80mm): escolher modo "Compacto"
  □ Se laser/jato: modo "Normal"
  □ Imprimir teste

□ Menu → Configurações → Taxas de Pagamento
  □ Configurar taxa de PIX (normalmente 0%)
  □ Configurar taxa de Débito (ex: 1,99%)
  □ Configurar taxas de Crédito:
    - 1x: _____% (ex: 3,49%)
    - 2x: _____% (ex: 4,99%)
    - 3x: _____% (ex: 5,49%)
    - 6x: _____% (ex: 6,99%)
    - 12x: _____% (ex: 8,99%)
  □ Salvar
```

### ✅ 3. Criar Usuários

```
□ Menu → Usuários → Novo Usuário
  
  Usuário 1 (Proprietário/Gerente):
  □ Nome: _____________________________
  □ Email: _____________________________
  □ Senha: _____________________________ (anotar)
  □ Papel: Admin
  □ Salvar
  
  Usuário 2 (Atendente - se houver):
  □ Nome: _____________________________
  □ Email: _____________________________
  □ Senha: _____________________________ (anotar)
  □ Papel: Atendente
  □ Salvar
  
  Usuário 3 (Técnico - se houver):
  □ Nome: _____________________________
  □ Email: _____________________________
  □ Senha: _____________________________ (anotar)
  □ Papel: Técnico
  □ Salvar

□ Desativar usuário "admin" padrão (segurança)
```

### ✅ 4. Cadastro Inicial de Dados

```
□ Cadastrar 3-5 produtos mais vendidos:
  Produto 1:
  □ Nome: _____________________________
  □ Código/SKU: ________________________
  □ Preço Venda: R$ ____________________
  □ Preço Custo: R$ ____________________
  □ Estoque: __________________________
  □ Salvar
  
  (Repetir para outros produtos)

□ Cadastrar 2-3 clientes frequentes (opcional):
  Cliente 1:
  □ Nome: _____________________________
  □ Telefone: __________________________
  □ Salvar
```

---

## 📋 FASE 3: TESTE FUNCIONAL (15-20 min)

### ✅ Testar Funcionalidades Principais

```
□ TESTE 1: Fazer uma venda
  □ Nova Venda
  □ Adicionar produto
  □ Escolher forma de pagamento
  □ Finalizar
  □ Verificar se aparece na lista
  □ Imprimir recibo (teste de impressão)
  □ ✅ Funcionou

□ TESTE 2: Criar uma OS
  □ Nova OS
  □ Preencher dados do cliente
  □ Preencher dados do equipamento
  □ Definir valores
  □ Salvar
  □ Verificar se aparece na lista
  □ Imprimir OS (teste)
  □ ✅ Funcionou

□ TESTE 3: Verificar Financeiro
  □ Ir em Fluxo de Caixa
  □ Confirmar que venda aparece
  □ Verificar valor correto
  □ ✅ Funcionou

□ TESTE 4: Relatórios
  □ Ir em Relatórios
  □ Escolher período "Hoje"
  □ Confirmar que venda aparece
  □ Verificar totais
  □ ✅ Funcionou

□ TESTE 5: Backup
  □ Ir em Backup
  □ Gerar Backup
  □ Baixar arquivo
  □ Guardar em local seguro
  □ ✅ Funcionou
```

---

## 📋 FASE 4: TREINAMENTO DO CLIENTE (45-60 min)

### ✅ Treinamento Prático

```
□ MÓDULO 1: Vendas (15 min)
  □ Demonstrar venda completa
  □ Cliente faz venda supervisionado
  □ Explicar:
    - Como buscar produtos
    - Como aplicar desconto
    - Diferença entre formas de pagamento
    - Como imprimir recibo
  □ ✅ Cliente consegue fazer sozinho

□ MÓDULO 2: Ordens de Serviço (15 min)
  □ Demonstrar criação de OS
  □ Cliente cria OS supervisionado
  □ Explicar:
    - Como preencher dados
    - Como atualizar status
    - Como finalizar e receber
  □ ✅ Cliente consegue fazer sozinho

□ MÓDULO 3: Cadastros (10 min)
  □ Mostrar como cadastrar cliente
  □ Mostrar como cadastrar produto
  □ Cliente cadastra 1 produto sozinho
  □ ✅ Cliente entendeu

□ MÓDULO 4: Relatórios (10 min)
  □ Mostrar Painel
  □ Mostrar Relatórios
  □ Mostrar Fluxo de Caixa
  □ Explicar principais métricas
  □ ✅ Cliente entendeu

□ MÓDULO 5: Backup (5 min)
  □ Demonstrar backup
  □ Cliente faz backup supervisionado
  □ Explicar importância
  □ Agendar backup diário
  □ ✅ Cliente sabe fazer backup
```

---

## 📋 FASE 5: ENTREGA E DOCUMENTAÇÃO (10-15 min)

### ✅ Finalização

```
□ Entregar materiais:
  □ Manual do usuário (impresso ou PDF)
  □ FAQ (impresso ou PDF)
  □ Cartão com contato de suporte
  □ Link dos vídeos tutoriais

□ Anotar credenciais do cliente:
  □ URL: _____________________________
  □ Usuário: __________________________

□ Configurar atalho na área de trabalho (opcional)

□ Salvar backup inicial em local seguro

□ Assinar termo de entrega/instalação
```

## 🚨 CHECKLIST DE PROBLEMAS COMUNS

### ✅ Solução de Problemas Durante Instalação

```
□ PROBLEMA: Internet lenta/instável
  □ Solução: Sistema funciona offline, explicar isso
  □ ✅ Resolvido

□ PROBLEMA: Impressora não funciona
  □ Solução 1: Testar impressão do Windows
  □ Solução 2: Trocar modo (Normal/Compacto)
  □ Solução 3: Verificar drivers
  □ ✅ Resolvido

□ PROBLEMA: Cliente esqueceu senha
  □ Solução: Redefinir pelo admin
  □ ✅ Resolvido

□ PROBLEMA: Navegador desatualizado
  □ Solução: Atualizar navegador ou usar Chrome
  □ ✅ Resolvido

□ PROBLEMA: Computador muito lento
  □ Solução: Fechar programas desnecessários
  □ Solução 2: Recomendar upgrade (mais RAM)
  □ ✅ Resolvido / Orientado
```
