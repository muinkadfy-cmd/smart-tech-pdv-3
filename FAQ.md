## 🔐 LOGIN E ACESSO

### 1. Esqueci minha senha. O que faço?

**R:** Apenas o administrador pode redefinir senhas. Entre em contato com o responsável pelo sistema ou com o suporte técnico.

### 2. Qual é o usuário e senha padrão?

**R:** 
- **Usuário:** `admin`
- **Senha:** `admin123`

⚠️ **IMPORTANTE:** Troque imediatamente após primeiro acesso!

### 3. Posso usar o mesmo login em vários computadores?

**R:** Sim, mas não simultaneamente. Se fizer login em outro computador, o anterior será desconectado.

### 4. O sistema funciona em celular?

**R:** Sim! Acesse pelo navegador do celular. O sistema se adapta automaticamente (responsivo).

### 5. Preciso de internet para usar?

**R:** Não! O sistema funciona **offline**. Apenas algumas funções (WhatsApp, sincronização na nuvem) precisam de internet.

## 🛒 VENDAS

### 6. Como fazer uma venda rápida sem cadastrar cliente?

**R:** 
1. Menu → Vendas → Nova Venda
2. Deixe o campo "Cliente" vazio
3. Adicione produtos e finalize

### 7. Posso vender produtos sem cadastrá-los?

**R:** Sim! Use a opção **"Produto Avulso"** ao adicionar item na venda. Digite nome e valor na hora.

### 8. Como aplicar desconto?

**R:** Na tela de nova venda:
1. Adicione os produtos
2. No campo "Desconto", digite o valor
3. Escolha entre R$ (valor fixo) ou % (porcentagem)

### 9. Como fazer venda parcelada?

**R:** 
1. Ao finalizar venda, escolha forma de pagamento "Crédito"
2. Selecione número de parcelas (1x a 12x)
3. Sistema calcula automaticamente as taxas

### 10. Posso cancelar uma venda?

**R:** No momento, não há função de cancelamento direto. Entre em contato com o suporte para reverter uma venda.

### 11. Como enviar recibo por WhatsApp?

**R:** 
1. Ao finalizar venda, clique no botão verde "WhatsApp"
2. Ou na lista de vendas, clique nos 3 pontinhos → "Enviar WhatsApp"
3. Confirme o número do cliente

**Requisito:** Cliente precisa ter telefone cadastrado.

### 12. O estoque diminui automaticamente ao vender?

**R:** Sim! Ao finalizar uma venda, o estoque é atualizado automaticamente.

## 🔧 ORDENS DE SERVIÇO

### 13. Como criar uma OS?

**R:** Menu → Ordens de Serviço → Nova OS → Preencha os dados → Salvar

### 14. Posso editar uma OS depois de criada?

**R:** Sim! Clique na OS na lista → Editar → Faça alterações → Salvar

### 15. Como finalizar uma OS?

**R:** 
1. Abra a OS
2. Edite
3. Mude status para "Pronta"
4. Quando cliente buscar: mude para "Entregue"
5. Marque pagamento como "Pago"

### 16. A OS entra no financeiro automaticamente?

**R:** Sim! Quando você marca o pagamento como "Pago", o sistema registra automaticamente no fluxo de caixa.

### 17. Como imprimir a OS para o cliente assinar?

**R:** Ao criar a OS, clique no botão "Imprimir OS". Ou na lista, clique nos 3 pontinhos → "Imprimir".

### 18. Posso adicionar fotos do defeito na OS?

**R:** No momento, fotos só estão disponíveis para aparelhos usados. Para OS, use o campo "Observações" para descrever.

## 👥 CLIENTES

### 19. É obrigatório cadastrar cliente?

**R:** Não! Você pode fazer vendas e OS sem cliente cadastrado. Mas cadastrar ajuda a:
- Enviar recibo por WhatsApp
- Ver histórico de compras
- Ter controle de cobranças

### 20. Como buscar um cliente?

**R:** Use a barra de busca e digite:
- Nome (parcial ou completo)
- Telefone
- CPF

### 21. Posso editar dados de um cliente?

**R:** Sim! Clique no cliente → Editar → Altere → Salvar

### 22. Como ver histórico de um cliente?

**R:** Clique no cliente na lista. Você verá todas as vendas e OS dele.

### 23. Posso excluir um cliente?

**R:** Sim, mas apenas se ele não tiver vendas ou OS vinculadas.

## 📦 PRODUTOS E ESTOQUE

### 24. Como cadastrar um produto?

**R:** Menu → Produtos → Novo Produto → Preencha → Salvar

**Campos obrigatórios:** Nome e Preço de Venda

### 25. Preciso cadastrar código/SKU?

**R:** Não é obrigatório, mas ajuda a:
- Encontrar produto mais rápido
- Usar leitor de código de barras (futuro)

### 26. Como atualizar estoque?

**R:** 
- **Manualmente:** Edite o produto e altere quantidade
- **Automaticamente:** Estoque diminui ao vender

### 27. Como saber quando produto está acabando?

**R:** Configure "Estoque Mínimo" no produto. O sistema alertará quando quantidade ficar abaixo do mínimo.

### 28. Estoque ficou negativo, o que fazer?

**R:** 
1. Vá em Produtos
2. Edite o produto
3. Corrija a quantidade manualmente

Estoque negativo significa que vendeu mais do que tinha registrado.

### 29. Posso cadastrar serviços como produtos?

**R:** Sim! Cadastre serviços (ex: "Troca de Tela") como produtos sem controle de estoque.

## 📱 USADOS

### 30. Como registrar compra de um aparelho usado?

**R:** Menu → Compra Usados → Nova Compra → Preencha dados → Adicione fotos → Salvar

**O que acontece:** Aparelho entra no estoque e sistema registra saída financeira automática.

### 31. Posso adicionar fotos do aparelho?

**R:** Sim! É **altamente recomendado**. Tire fotos:
- Frente
- Traseira
- Tela ligada
- Defeitos (se houver)

### 32. Como vender um aparelho usado?

**R:** Menu → Venda Usados → Selecione aparelho → Preencha valor → Registrar Venda

**O que acontece:** Aparelho sai do estoque, sistema registra entrada financeira e calcula lucro automaticamente.

### 33. Como ver fotos dos aparelhos em estoque?

**R:** Na lista "Em estoque", clique em **"📸 Ver fotos (N)"**. Abre galeria com todas as fotos.

### 34. Como sei se tive lucro ou prejuízo?

**R:** No fluxo de caixa, a descrição da venda mostra:
- "Lucro: R$ XXX" (se vendeu por mais)
- "Prejuízo: R$ XXX" (se vendeu por menos)

### 35. IMEI é obrigatório?

**R:** É altamente recomendado para:
- Rastreabilidade
- Garantia
- Prova em caso de problemas

## 💰 FINANCEIRO

### 36. O que é "Total Bruto"?

**R:** Soma de todas as vendas SEM descontar taxas ou descontos.

### 37. O que é "Total Líquido"?

**R:** Dinheiro que realmente entra na conta, JÁ descontadas as taxas de cartão.

### 38. O que é "Lucro Bruto"?

**R:** Venda menos custo do produto (sem descontar taxas).

### 39. O que é "Lucro Líquido"?

**R:** Lucro real, já descontadas as taxas de cartão e operadora.

### 40. Como adicionar uma despesa (gasto)?

**R:** Menu → Fluxo de Caixa → Nova Movimentação → Tipo: Saída → Preencha → Salvar

### 41. Como ver quanto faturei no mês?

**R:** Menu → Relatórios → Escolha período "Mês Atual" → Veja "Total Bruto" ou "Total Líquido"

### 42. As taxas de cartão são calculadas automaticamente?

**R:** Sim! Configure as taxas em Configurações → Taxas de Pagamento. O sistema calcula automaticamente.

### 43. Por que meu lucro é diferente do faturamento?

**R:** Porque:
- **Faturamento** = Total de vendas
- **Lucro** = Faturamento - Custos - Taxas - Descontos

## 🖨️ IMPRESSÃO

### 44. Como imprimir um recibo?

**R:** Ao finalizar venda, clique em "Imprimir". Ou na lista de vendas, clique nos 3 pontinhos → "Imprimir".

### 45. Como imprimir uma OS?

**R:** Ao criar OS, clique em "Imprimir OS". Ou na lista, 3 pontinhos → "Imprimir".

### 46. Impressão sai cortada ou errada, o que fazer?

**R:** 
1. Vá em Configurações → Impressora
2. Escolha modo "Compacto" (para impressoras térmicas 80mm)
3. Teste novamente

### 47. Como desativar termos de garantia na impressão?

**R:** Configurações → Termos de Garantia → Desmarque "Habilitar exibição"

### 48. Posso personalizar o que aparece no recibo?

**R:** Sim! Edite os dados da empresa em Configurações → Empresa. Nome, telefone e endereço aparecem no recibo.

## 💾 BACKUP

### 49. Com que frequência devo fazer backup?

**R:** **Diariamente**, no final do expediente. Ou antes de:
- Atualizar o sistema
- Fazer manutenção no computador
- Qualquer operação de risco

### 50. Onde salvar o backup?

**R:** 
✅ Pen drive  
✅ Nuvem (Google Drive, Dropbox)  
✅ Email para você mesmo  
✅ HD externo  

❌ Apenas no mesmo computador (não protege se ele quebrar)

### 51. Como restaurar um backup?

**R:** Menu → Backup → Restaurar Backup → Selecione arquivo → Confirme

⚠️ **ATENÇÃO:** Restaurar sobrescreve todos os dados atuais!

### 52. Backup funciona entre computadores diferentes?

**R:** Sim! Faça backup em um computador e restaure em outro.

### 53. O que o backup salva?

**R:** **TUDO:**
- Vendas
- Ordens de Serviço
- Clientes
- Produtos
- Usados
- Movimentações financeiras
- Configurações

## ⚙️ CONFIGURAÇÕES

### 54. Como mudar o nome da minha empresa que aparece no recibo?

**R:** Configurações → Empresa → Edite "Nome Fantasia" → Salvar

### 55. Como adicionar meu telefone no recibo?

**R:** Configurações → Empresa → Preencha "Telefone" → Salvar

### 56. Como configurar taxas de cartão?

**R:** Configurações → Taxas de Pagamento → Edite valores → Salvar

**Exemplos comuns:**
- PIX: 0%
- Débito: 1,99%
- Crédito 1x: 3,49%
- Crédito 12x: 8,99%

### 57. Como criar um novo usuário?

**R:** Menu → Usuários → Novo Usuário → Preencha → Escolha papel → Salvar

**Somente administradores** podem criar usuários.

### 58. Qual a diferença entre Admin, Atendente e Técnico?

**R:** 
- **Admin:** Acesso total (tudo)
- **Atendente:** Vendas, clientes, produtos (sem financeiro/config)
- **Técnico:** Ordens de serviço e clientes (sem vendas/financeiro)

### 59. Posso ter mais de uma loja no sistema?

**R:** Sim! O sistema é multi-loja (multi-tenant). Cada loja tem seus próprios dados isolados.

## 🐛 PROBLEMAS TÉCNICOS

### 60. Sistema está lento, o que fazer?

**R:** 
1. Feche outras abas do navegador
2. Reinicie o navegador
3. Limpe o cache (Ctrl + Shift + Del)
4. Reinicie o computador

### 61. Vendas não aparecem nos relatórios

**R:** 
1. Verifique o filtro de período (mude para "Todos")
2. Confirme a data da venda
3. Recarregue a página

### 62. Não consigo fazer login

**R:** 
1. Verifique usuário e senha (são sensíveis a maiúsculas)
2. Tente limpar cache do navegador
3. Entre em contato com suporte

### 63. Impressão não funciona

**R:** 
1. Verifique se impressora está ligada
2. Teste imprimir página do navegador
3. Use modo "Compacto" para térmicas
4. Selecione impressora correta na janela de impressão

### 64. Sistema não sincroniza com nuvem

**R:** 
1. Verifique conexão com internet
2. Aguarde alguns minutos (sincroniza automaticamente)
3. Recarregue a página

### 65. Tela fica em branco ao abrir

**R:** 
1. Recarregue a página (F5)
2. Limpe cache do navegador
3. Tente outro navegador (Chrome, Edge, Firefox)
